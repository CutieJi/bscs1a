System Guide

MISLend Documentation

Learn the full student and admin workflow with a cleaner guide layout that now matches your home page color palette. This includes borrowing, returning, approvals, exports, and demo testing accounts.

Student Guide

  • 1) Login: Access your dashboard using your student email or ID.
  • 2) Account Status: If your account is pending, wait for an administrator to verify and approve your registration.
  • 3) Find Equipment: Use the Browse Equipment section to search for available tools. You can filter by category or status.
  • 4) Borrowing: Click Borrow on an available item. Fill in the Room Number and your expected Return Time.
  • 5) Confirmation: Once submitted, the item will appear in your My Borrowed list.
  • 6) Returning: Locate the item in My Borrowed and click Return. Select the item's condition.
    - Good: Item returns to available status immediately.
    - Not Good / Damaged: Item is flagged for maintenance.
  • 7) QR Convenience: Use the Scan feature for rapid borrowing and returning by scanning the physical QR code on the equipment.

Admin Guide

  • 1) System Overview: The main dashboard provides live counts of total, borrowed, and maintenance equipment.
  • 2) Equipment Management:
    - Add: Create new equipment entries with unique IDs.
    - Edit: Update names, IDs, or manually change status.
    - QR: Generate and download QR codes for physical tagging.
  • 3) User Approvals: Navigate to Users → Pending to review registrations. Approve legitimate students or reject invalid requests.
  • 4) Data Export: Use Logs to filter borrowing history and export to Excel/CSV for reporting and compliance audits.

Quick Tips & Best Practices

  • Equipment Care: Always verify cables and accessories are included before returning an item.
  • Reporting: If an item is damaged, use the Not Good status during return and leave a brief note if prompted.
  • Scanner Setup: Ensure you are using HTTPS for the QR scanner to function on mobile devices.
  • Maintenance: Admin should regularly check items in the maintenance queue.
