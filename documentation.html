Step-by-step guide on how to use MIS eBorrow (Student + Admin). Includes common demo accounts and quick tips.

📌 Student Guide

  • 1) Login using your student account.
  • 2) If status is pending, wait for admin approval.
  • 3) Open Browse Equipment and search/filter items.
  • 4) Click Borrow on available equipment.
  • 5) Fill required fields: Room and Return Time.
  • 6) Submit and check My Borrowed for active items.
  • 7) When returning, choose condition:
    ✅ Good → back to available
    ⚠️ Not Good / Damaged → becomes maintenance
  • 8) Optional: Use Scan to borrow/return using QR code.

🛠 Admin Guide

  • 1) Login using an admin account.
  • 2) Dashboard shows: equipment totals, borrowed badge, today borrows, users.
  • 3) Manage Equipment:
    ➕ Add equipment
    ✏️ Edit equipment
    🧾 Generate QR
    🗑 Delete (only if not borrowed)
  • 4) Pending approvals:
    User Management → Pending → Approve or Reject
  • 5) Borrowing Logs:
    Filter by date / student / equipment then export CSV.

✅ Equipment Condition Rules

If an item was returned and marked Not Good, it should be set to maintenance.

  • After repair, admin can edit equipment status and set it back to available.
  • Only available items are shown to students by default.
  • QR scanning requires HTTPS or localhost.
