EquipLend Documentation

Quick guide on how to use the system (Student & Admin).

Last updated: 2026

Student Guide

1) Log in

  • Open the website and log in using your student account.
  • If your account is new, it may appear as Pending until approved by admin.

2) Browse equipment

  • Go to Browse Equipment.
  • Use filters: Category, Status, and Search bar.
  • Only Available items can be borrowed.

3) Borrow equipment

  • Click Borrow on an equipment card.
  • Fill up:
    • Room (required)
    • Return time (required)
    • Purpose (optional)
  • Click Confirm Borrow.

4) Return equipment

  • Go to My Borrowed tab.
  • Click Return Equipment.
  • Choose item condition:
    • Good → becomes available
    • Damaged → becomes maintenance
  • Optional: add notes.

5) Scan QR

  • Go to Scan.
  • Allow camera permission (must be HTTPS or localhost).
  • Scan equipment QR to borrow/return faster.
  • Manual entry is available if camera is blocked.

Admin Guide

1) Dashboard

  • Shows total equipment, available, borrowed, today borrows, and total users.
  • Check Recent Activities and Overdue Returns.

2) Manage Equipment

  • Add new equipment with unique Equipment ID.
  • Edit equipment details (name, category, description).
  • Generate QR code for any equipment.
  • Delete equipment only if it’s not currently borrowed.

3) Edit Condition / Status

  • In Edit Equipment modal, admin can set status:
    • available (ready to borrow)
    • maintenance (not good / under repair)
    • borrowed (manual override, optional)
  • If set to available, equipment is usable again.

4) User Management

  • Approve or reject pending accounts.
  • Add new users (Student/Admin).
  • Edit user info (name, email, role, student ID).
  • Delete user from database (note: this may not remove Firebase Auth unless you use Admin SDK).

5) Logs

  • Filter borrowing logs by date/student/equipment.
  • Export logs to CSV.

Troubleshooting

  • Camera not working: Use HTTPS (or localhost) + allow permissions.
  • Borrow button disabled: Item is not available.
  • Overdue showing: Expected return date/time is already passed.
