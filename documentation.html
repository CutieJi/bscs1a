📘 Documentation

Quick guide on how to use 1AFeedbackHub—submit feedback, track submissions, and manage data (admin).

🧑‍🎓 Student Guide

  • Submit Feedback: Choose category, type, priority, then write your message.
  • Anonymous Option: Enable “Submit anonymously” if you don’t want your name shown.
  • My Submissions: View your history and track status: pending → reviewed → resolved.

🛡 Admin Guide

  • Overview: See totals and recent feedback.
  • All Feedback: Filter by status/category/type + search.
  • Manage Users: Add, edit user info (name/student ID).
  • Analytics: View trends per category/type/priority.
Tip: Press Ctrl + R to refresh if data is missing. © 2026 1AFeedbackHub
