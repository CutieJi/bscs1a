🔒 Privacy Policy

We respect your privacy. This page explains what data is collected and how it’s used.

📌 Data Collected

  • Account: name, email, student ID (if provided)
  • Feedback: category, type, priority, message, timestamps

🧾 How We Use Data

  • Improve services: analyze feedback trends
  • Track status: pending/reviewed/resolved updates
  • Security: access controlled using Firebase Authentication & Firestore Rules

🛡 Security

Your data is stored in Firebase Firestore. Access is restricted by authentication rules. Anonymous feedback hides your identity in the UI when enabled.
