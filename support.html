Support

Need help? Here are quick fixes and where to report issues.

🧩 Common Issues

  • Camera not working (QR Scan): Use HTTPS or localhost. Allow camera permission in browser settings.
  • Account is pending: Wait for admin approval (User Management → Pending).
  • Cannot borrow equipment: Item may be borrowed or in maintenance.
  • Wrong return status: Admin can update equipment status in Manage Equipment.

📩 Contact Admin

For support requests, contact the system administrator.

  • Email: admin@cs1a.com
  • Office: MIS Office
  • Response Time: 24–48 hours

📝 How to Report a Bug

  • Step 1: Describe what you were doing.
  • Step 2: Include the exact error message (if any).
  • Step 3: Add screenshots if possible.
  • Step 4: Mention device + browser (Chrome/Edge/Mobile).

✅ Quick Tips

  • Refresh dashboard if numbers look delayed.
  • Check Borrowing Logs for record verification.
  • If item is fixed, set status back to available.
