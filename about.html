Simple Process

How It Works

1

Scan QR Code

Students scan the QR code attached to any equipment using their camera. The system instantly identifies the item and pulls its availability status.

2

Send Request

Students select an expected return date and provide a purpose. The borrow request is instantly queued for admin approval.

3

Admin Approval

Admins review all pending requests on their dashboard and approve constraints. A real-time notification loop keeps everyone informed.

4

Return & Evaluate

Once finished, students return the item and rate its condition. Admins finalize the transaction, freeing the item for the next user.
